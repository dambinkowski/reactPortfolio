Calculator

A React Native calculator app with arithmetic operations, chained calculations, sign toggle, and percent handling.

React Native State management Touchable controls

Report Summary

This project implements a mobile calculator interface using React Native state to manage the current value, selected operation, and intermediate results across chained calculations.

  • Supports addition, subtraction, multiplication, and division with live display updates.
  • Includes clear, sign-toggle, and percent controls to match expected calculator behavior.
  • Uses touch-friendly button grouping and visual contrast for fast one-hand use.
