Workout App

A workout timer and routine editor app with multiple screens, exercise sequencing, and data persistence.

React Native AsyncStorage Timer workflows

Report Summary

This project manages workout sessions with configurable routines, timed intervals, and persistent local storage so users can resume training plans between app launches.

  • Supports creating and editing exercise sequences with per-exercise durations.
  • Uses timer-driven state transitions to move through sets, rest windows, and completion.
  • Saves routines with AsyncStorage to preserve workout data on the device.
