Project 5

Workout App

A workout timer and routine editor app with multiple screens, exercise sequencing, and data persistence.

React Native AsyncStorage Timer workflows

Report Summary

This project implements a multi-screen workout assistant where users can create routines, edit exercises, and run guided timers for each session.

  • Home screen displays available workouts and supports creating new routines.
  • Workout screen runs interval timers and shows upcoming exercises.
  • Edit screen updates exercise names/times and persists data with AsyncStorage.
