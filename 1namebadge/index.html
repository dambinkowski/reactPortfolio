Name Badge

Shows your name on the phone.


Report Summary

This project builds a simple React Native name badge screen with strong visual hierarchy. The layout uses a bold background, oversized heading text, and a contrasting name strip to keep the identity message clear.

  • Uses a full-screen red stage to give the badge a high-impact, poster-style look.
  • Stacks greeting text above a white name panel to create clear reading order.
  • Applies large uppercase type and centered alignment for quick readability in landscape mode.
